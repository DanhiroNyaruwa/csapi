Traffic Data API Documentation

This is an experimental web service allowing users to navigate and explore traffic statistics for major roads. Sample data used covers the South West region of England

QueryRequest sample
Get traffic count data for all count points https://cstrafficapi.herokuapp.com/api/aadfs
Get traffic count stats for a known count point https://cstrafficapi.herokuapp.com/api/aadfs?CP=6023
Get traffic count stats by road https://cstrafficapi.herokuapp.com/api/aadfs?Road=M5
Get traffic count stats by roadcategory https://cstrafficapi.herokuapp.com/api/aadfs?RoadCategory=TM
Get traffic count stats by Year https://cstrafficapi.herokuapp.com/api/aadfs?AADFYEAR=2015
Get traffic count stats by LocalAuthority https://cstrafficapi.herokuapp.com/api/aadfs?LocalAuthority=Devon
Filtering
API allows filtering data by using the following categories:
  • CP (numeric count point identifier)
  • AADFYEAR (Year traffic counts were made)
  • Region
  • Road
  • RoadCategory
  • LocalAuthority

In addition to the filters applied above, data can be further filtered by combining the criteria. For Instance To query stats by year and road we can use the request format below

https://cstrafficapi.herokuapp.com/api/aadfs?Road=M5&AADFYEAR=2015"
